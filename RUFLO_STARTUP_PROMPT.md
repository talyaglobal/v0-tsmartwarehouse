# Ruflo v3.5 Master Prompt — Warebnb v1 (Production)

## Project Overview
**Warebnb** - The Marketplace for Warehouse Space (similar to Airbnb for warehouses)

**Tech Stack**: Next.js 16, React 19, TypeScript, Prisma, Supabase, Stripe, Firebase, Tailwind CSS, Radix UI

**Core Features**:
- Warehouse marketplace platform
- Booking system with 10% deposits
- Pallet check-in/check-out with QR codes and photo verification
- Real-time chat between clients and warehouse staff
- Payment processing and invoicing
- Admin dashboard and warehouse management
- Real-time notifications and calendar integration
- 3D warehouse visualization

